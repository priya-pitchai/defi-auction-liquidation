# Auction-Based DeFi Lending Protocol

A decentralized lending protocol built with Solidity and Hardhat.

## Features

- Multiple lenders contribute to a shared liquidity pool.
- Borrowers lock ETH as collateral.
- Loans are issued in stablecoins.
- Chainlink ETH/USD Price Oracle.
- Fixed interest rate.
- Automated liquidation using Keepers.
- Auction-based collateral liquidation.
- Surplus auction proceeds returned to the borrower.
